## World Cup 2026 Draw App

Interactive World Cup group draw simulator built with **React**, **Vite**, and **Tailwind CSS**.

</div>

This app lets you create a World Cup-style draw with animated transitions and two game modes.

## What it does

- Add participants manually
- Preview the 12 groups before starting
- Run an animated draw
- Show the final results by participant
- Restart the draw at any time

## Draw modes

- **One Group Per Person**  
  Each participant gets one group first. If there are groups left, the app asks for volunteers to receive extra groups.

- **Equal Distribution**  
  The 12 groups are distributed as evenly as possible across all participants.

## Built with

- React
- Vite
- Tailwind CSS
